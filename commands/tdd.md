description: Start the Sixth Sense TDD workflow (Red-Green-Refactor).
---
Use the `Skill` tool with `skill: "sixth-sense:test-driven-development"` (or, on Antigravity, `activate_skill` with `test-driven-development`) to load and run the TDD skill — use this exact qualified name, a bare "tdd" may not resolve to it. Follow the Red-Green-Refactor cycle strictly, alternating between a failing test, minimal passing code, and active refactoring.
