description: Synthesize the current conversation into a structured spec document.
---
Use the `Skill` tool with `skill: "sixth-sense:synthesizing-spec"` (or, on Antigravity, `activate_skill` with `synthesizing-spec`) to load and run the spec-synthesis skill — use this exact qualified name, a bare "to-spec" may not resolve to it. It reviews the session history and produces a spec in the standard format.
