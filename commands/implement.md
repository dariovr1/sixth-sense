Use the `Skill` tool with `skill: "sixth-sense:implementing-work"` (or, on Antigravity, `activate_skill` with `implementing-work`) to load and run the implementation-workflow skill — use this exact qualified name, a bare "implement" may not resolve to it. It guides you through breaking down the task, applying vertical-slice TDD, and verifying the result.
